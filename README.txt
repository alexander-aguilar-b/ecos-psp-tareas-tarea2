*********************************************************************************************************************************************************		  
1-Acceder a la siguiente URL del repositorio en github: https://github.com/alexander-aguilar-b/ecos-psp-tareas-tarea2

3-Descomprimir el archivo anterior.
4-Abrir una ventana de comandos de windows dentro de la carpeta descomprimida en el paso anterior (por ejemplo en C:\Users\edgaguil\Downloads\ecos-psp-tareas-tarea2-master)

 el número de clases, número de métodos y número de lineas de código.
